I created a program that would generate random mazes using a depth first recursive algorithm. Afterward, I built a program to solve the maze using depth first recurison as well and this maze is displayed on a graphical user interface (GUI) that will animate the process.

I created a AI to play the game Othello against other users and AI's. In this project, I had to use Binary Search Trees to look at all possible outcomes of the opponent's move. Afterward, my AI would choose the move that most likely resulting in victory.
